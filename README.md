# odin-notes-tw

The Odin Project（Foundations + Full Stack JavaScript）的繁體中文自足講義。
Markdown in, MkDocs Material out，手機直接讀網站。

## 快速開始

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
```

內容生成：開 Claude Code，說「讀 CLAUDE.md，跑 Bootstrap」。
之後每課丟 lesson URL 即可（`odin-lecture` skill 會接手）。
