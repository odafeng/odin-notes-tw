# Odin 講義

**The Odin Project 的繁體中文自足講義。** 每一課都把官方課文與指定閱讀消化改寫成連貫的中文內容——讀完一課，不用點任何外部連結，就能學會。範圍鎖定 **Foundations** 與 **Full Stack JavaScript** 兩條路線。
