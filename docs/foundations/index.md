The Odin Project **Foundations** path 的繁體中文自足講義總覽。Foundations 是整條學習路線的起點，從 web 如何運作、命令列與 Git，一路帶到 HTML、CSS、Flexbox 與 JavaScript 基礎——共一門課、46 課。
